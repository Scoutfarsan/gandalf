## Releases / Versioning

- **v7.0** = nuvarande baseline (ops + curated systemd + deploy)
- **v3.0** = tidigare baseline-tag som **ersatts av v7.0** (rebrand).  
  Taggen ligger kvar för historik/spårbarhet men ska inte användas för ny baselining.

**Policy:**
- Använd alltid **senaste major-taggen** (ex. v7.x) när du baselinar eller deployar.
- Legacy-innehåll ligger under **/old** och ändras endast vid behov.
